# HealthCare_StrokeAnalysis
An AI driver application designed to predict the probability of stroke occurrence based on health data using the "Logistic Regression" algorithm << Machine Learning >>. This project features advanced data preprocessing and a "Flask" web interface.


<< - >> How to run it:
1. Clone the repo: `git clone <https://github.com/Shaimaa26/HealthCare_StrokeAnalysis/tree/main>`
2. Install libraries: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open `http://127.0.0.1:5000` in your browser.
